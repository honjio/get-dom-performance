DOM 情報の取得と変更スピードを比較

結果は console でご確認ください

特定の DOM 情報を取得及び変更する

下記コードをそれぞれ for 文で 50000 回実行して計測

                    
    // ClassName で取得して text を変更
    document.getElementsByClassName('js-item')[0].textContent = i;

    // セレクターで取得して text を変更
    document.querySelector('.js-item').textContent = i;

    // 取得したい DOM の親（変数保持）から セレクターで取得して text を変更
    parent.querySelector('.js-item').textContent = i;

    // id で取得して text を変更
    document.getElementById('js-item').textContent = i;

    // 変数保持 で text を変更
    item.textContent = i;

同じ class 名を持つ全ての DOM 情報を取得する

下記コードをそれぞれ for 文で 50000 回実行して計測

                        
    // ClassName で取得
    document.getElementsByClassName('js-item');

    // 取得したい複数の DOM の親（変数保持）から ClassName で取得
    parent.getElementsByClassName('js-item');

    // セレクターで取得
    document.querySelectorAll('.js-item');

    // 取得したい複数の DOM の親（変数保持）からセレクターで取得
    parent.querySelectorAll('.js-item');
